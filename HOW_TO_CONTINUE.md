## 🛠️ Настройка окружения (Android Studio)

### 1. Открыть проект
```
File → Open → /mnt/okcomputer/output/AutoDiagAI
```

### 2. Дождаться индексации Gradle
- Жди 5-10 минут (первый запуск)
- Android Studio скачает зависимости

### 3. Проверить сборку
```
Build → Make Project (Ctrl+F9)
```

### 4. Настроить эмулятор
```
Tools → Device Manager → Create Device
→ Выбери: Pixel 6, API 33
```

### 5. Запустить
```
Run → Run 'app' (Shift+F10)
```

## 📞 Если что-то не работает

**Проблема:** Gradle не собирается  
**Решение:** `File → Invalidate Caches → Invalidate and Restart`

**Проблема:** Нет Android SDK  
**Решение:** `Tools → SDK Manager → Установи API 33`

**Проблема:** Ошибки в коде  
**Решение:** Загрузи файл с ошибкой в чат - я исправлю
