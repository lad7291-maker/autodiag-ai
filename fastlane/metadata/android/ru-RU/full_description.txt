AutoDiagAI - ваш интеллектуальный помощник для диагностики автомобиля через OBD2.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
• Считывание и сброс кодов ошибок (DTC)
• Мониторинг параметров двигателя в реальном времени
• ИИ-анализ данных для выявления проблем
• История диагностики с экспортом отчетов
• Поддержка всех стандартных OBD2 адаптеров (Bluetooth)
• Предиктивное техническое обслуживание
• Голосовое управление

БЕЗОПАСНОСТЬ И КОНФИДЕНЦИАЛЬНОСТЬ:
• Все данные хранятся локально на устройстве
• Нет передачи данных на серверы
• Надежное шифрование базы данных

ТРЕБОВАНИЯ:
• Android 8.0+ (API 26+)
• Bluetooth 4.0+ OBD2 адаптер
• Автомобиль с поддержкой OBD2 (большинство автомобилей с 1996 года)

Важно: Приложение предназначено для информационных целей. Для серьезных проблем всегда консультируйтесь с квалифицированным механиком.
